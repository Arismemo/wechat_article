## 2. 依赖拆解

### 2.1 Apple 侧依赖

- `快捷指令`
  - 作用：读取剪贴板、调用接入 API、展示回执
  - 状态：已确认可行
  - 风险：不同 iOS 版本和设备存在差异

- `Back Tap`
  - 作用：触发快捷指令
  - 状态：已确认可行
  - 风险：老设备或用户误触

### 2.2 微信侧依赖

- `公众号账号`
  - 已确认信息：名称 `奇点价值`，主体 `个人`，认证状态 `暂未认证`
  - 必须确认：草稿箱权限、素材权限、数据分析权限
  - 当前状态：部分确认

- `草稿箱接口`
  - 必须确认：字段约束、接口频率、错误码、是否对账号类型有限制
  - 当前状态：已通过 `draft/count` 真实接口验证

- `素材接口`
  - 必须确认：封面上传限制、正文图片上传策略、永久素材约束
  - 当前状态：已通过 `material/get_materialcount` 真实接口验证

- `图文分析数据`
  - 用途：阶段 6 历史表现反馈
  - 当前状态：可选，待确认
  - 兜底：后台手工录入发布表现数据

- `API IP 白名单`
  - 当前已配置：`117.72.155.136`、`222.79.178.127`
  - SSH / Tailscale 管理入口：`100.112.123.6`
  - 已确认公网出口：`117.72.155.136`
  - 当前状态：公网出口已包含在微信白名单中

### 2.3 基础设施依赖

- `PostgreSQL`
  - 用途：主业务数据、审计日志、版本记录
  - 状态：待部署

- `Redis`
  - 用途：任务队列、缓存、令牌缓存
  - 状态：待部署

- `对象存储`
  - 用途：原文快照、图片、导出 HTML
  - MVP 方案：服务器本机磁盘
  - 长期方案：可升级到 S3 兼容对象存储

- `HTTPS 域名`
  - 用途：快捷指令访问服务端
  - 状态：待部署

### 2.4 算法与模型依赖

- `LLM`
  - 用途：分析、写作、审稿、增长优化
  - 已确认：供应商为智谱，模型使用 `glm-5`
  - 已确认：Coding/Plan 基址为 `https://open.bigmodel.cn/api/coding/paas/v4`
  - 待确认项：预算、速率限制、上下文窗口、稳定性要求

- `搜索服务`
  - 用途：获取同题素材
  - 已确认：使用智谱 MCP `web_search_prime`
  - 配套读取：`web_reader`
  - 可选能力：视觉理解 `@z_ai/mcp-server`，代码阅读 `zread`
  - 待确认项：费用、接口限制、是否在服务端直接走 MCP

## 3. 阶段 0 必须确认的依赖

- 公众号账号及权限
- HTTPS 域名或临时回调地址
- 图文分析数据能力是否可用
