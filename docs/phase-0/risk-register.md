# 阶段 0 风险清单

更新时间：2026-03-07

## 1. 风险评级说明

- 影响：`高 / 中 / 低`
- 概率：`高 / 中 / 低`
- 处理策略：`规避 / 缓解 / 接受 / 转人工`

## 3. 当前最高优先级风险

- `R-001`：版权与平台风险
- `R-003`：公众号权限不完整
- `R-004`：IP 白名单问题
- `R-005`：抓取成功率不足
- `R-012`：知识未文档化
- `R-017`：账号权限不足

## 4. 阶段 0 风险处置动作

- 在阶段 1 开工前确认公众号权限和 IP 白名单流程。
- 在阶段 2 开工前确认抓取方案与人工补料兜底。
- 在阶段 4 开工前确认相似度门槛和人工审核规则。
- 在阶段 6 开工前确认分析数据来源和反馈闭环方案。

## 5. 更新规则

- 出现新的高风险假设时，必须先更新本文件，再推进实现。
- 任一风险从“待确认”变为“已缓解”时，必须记录对应措施。
